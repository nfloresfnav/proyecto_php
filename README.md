# Actividad 4 Proyecto Php

## Descripción
Este proyecto se ha realizado para aprender el uso de ubuntu e ir realizando una documentación y uso de git.

## Requisitos a instalar
* php -- sudo apt install php
* Apache2 - sudo apt install apache2
* dar los permisos necesarios a la carpeta /var/www/html


 ## Pasos para ejecución
 Una vez ya instalados los requisitos y haber dado los permisos en la carpeta, realizar las pruebas en navegador.
 Si esta usando alguna distribución con interfaz grafica, usar localhost.
 Si esta usando una distribución sin gui usar la ip de esta para realizar las pruebas.
